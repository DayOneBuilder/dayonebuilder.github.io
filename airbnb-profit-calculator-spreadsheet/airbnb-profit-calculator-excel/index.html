Airbnb profit calculator for Excel columns

If your Airbnb numbers already live in Excel, keep that file. This guide shows the column names the calculator recognizes and how to avoid the common percent/dollar mistakes when pasting rows.

Paste Excel rows into the calculator

Use it with

The browser calculator supports manual entry, pasted spreadsheet rows, automatic column mapping, local browser save, copy summary, and CSV export.

Accepted aliases

The import mapper accepts common labels such as ADR, nightly rate, occupancy, rent, mortgage, OTA fee, supplies, turnover, and furnishing cost.

Percent fields

Enter occupancy, platform fee, and tax reserve as normal percentages such as 62 or 8, not decimals such as 0.62 or 0.08.

Multiple properties

Each pasted row can be a different property. Use a property/name column if you want readable exported rows after calculation.
