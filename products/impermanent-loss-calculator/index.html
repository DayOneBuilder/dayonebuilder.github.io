Scope: 50/50 constant-product only. No concentrated liquidity, range exits, compounding, gas, or tax treatment.

Formula: IL% = 1 - (2 * sqrt(r) / (1 + r))

Where this helps

You want a fast before-entry check

Assumptions worth keeping in view

It is a 50/50 constant-product model

This is the classic LP math. If you are using concentrated liquidity or custom weights, use a more specific tool.

Fee input is manual on purpose

The calculator does not invent fee income for you. Put in the number you actually expect or already earned.

It keeps the dollar view simple

Token B is treated as the quote side for the value comparison so the hold and LP lines stay readable.

It does not touch gas or taxes

Those matter. They are separate from the hold-vs-LP comparison, so this page leaves them out instead of pretending otherwise.

This is a decision aid, not tax, legal, or yield advice. If it saves you from talking yourself into a bad LP entry, you can support future updates here. If not, keep using it.
