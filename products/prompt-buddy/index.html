Prompt Buddy

Free
Installable repo + optional support

Stop sending vague crypto prompts that force the agent to guess what you meant.

Prompt Buddy is a small prompt-coaching bundle for the moment before you hit send. It helps tighten token DD, wallet review, and yield-scan prompts so the agent gets a specific task instead of a fuzzy request.

The free core is intentionally narrow: check the prompt, spot what is missing, and rewrite it into something more usable.

Install with your agent

Tell your agent:
Install dayonebuilder/dob-prompt-buddy by Ben Ison.

Repo: github.com/DayOneBuilder/dob-prompt-buddy

Fallback if your agent does not infer the shell steps yet:

git clone https://github.com/DayOneBuilder/dob-prompt-buddy.git
cd dob-prompt-buddy
./install.sh

The install script adds reusable agent skills, copies the prompt-buddy assets into ~/.dayonebuilder/, and installs a helper launcher at ~/.local/bin/dob-prompt-buddy.

Token DD

Turns vague token questions into a real due-diligence ask with sections, checks, and evidence rules.

Wallet review

Forces wallet prompts to include chain, address, behavior checks, and limits on invented labels.

Yield scan

Pushes prompts beyond "find the best APY" and toward risk, liquidity, and sustainability questions.
