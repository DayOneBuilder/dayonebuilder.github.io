Free core first. Repo install when you want your agent to keep it.

DayOneBuilder publishes small crypto tools for messy token research, wallet tracking, and workflow cleanup. Every live product now has its own GitHub repo, its own install phrase, and its own free download.

That means you can still grab the file fast, but you can also tell your agent to install the repo and keep the product inside your workflow instead of losing it in another downloads folder.
